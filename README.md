# Carcassonne Hex Browser Game

Это браузерная игра "Каркассон" с шестиугольными плитками (гексами), написанная на Python с использованием библиотеки **Grafix** для генерации игровых ресурсов.

## Основные возможности

- **Авторизация**: Регистрация и вход пользователей с использованием Flask-Login.
- **Комнаты**: Создание игровых комнат для мультиплеера.
- **Гексагональная логика**: Использование осевых координат (axial coordinates) для управления игровым полем.
- **Серверная авторитетность**: Сервер управляет колодой, текущим игроком и очередностью ходов через Socket.IO.
- **Генерация ресурсов Grafix**: Игровые плитки (города, дороги, поля) процедурно отрисовываются с помощью библиотеки `grafix` и экспортируются в SVG.
- **Синхронизация в реальном времени**: Все действия игроков (поворот плитки, размещение на поле) мгновенно отображаются у всех участников в комнате.

## Требования

- Python 3.11+
- Библиотеки: Flask, Flask-SQLAlchemy, Flask-Login, Flask-SocketIO, grafix, numpy, shapely.

## Установка

1. Склонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd <repository-name>
   ```

2. Установите необходимые зависимости:
   ```bash
   pip install flask flask-sqlalchemy flask-login flask-socketio grafix numpy shapely
   ```

3. Сгенерируйте игровые плитки (SVG ассеты) с помощью библиотеки Grafix:
   ```bash
   python render_tiles.py
   ```
   Это создаст директорию `app/static/tiles/` и заполнит её SVG файлами для каждого типа плитки.

## Запуск игры

1. Запустите сервер разработки:
   ```bash
   python run.py
   ```

2. Откройте браузер и перейдите по адресу:
   `http://localhost:5000`

## Как играть

1. Зарегистрируйтесь и войдите в систему.
2. Создайте новую комнату или выберите существующую на панели управления.
3. В игровом интерфейсе вы увидите текущую плитку в правой панели ("Next Tile").
4. Используйте кнопку **Rotate**, чтобы повернуть плитку (правила Каркассон требуют совпадения сторон).
5. Нажмите на игровое поле, чтобы разместить плитку. Сервер проверит правильность размещения (например, город должен примыкать к городу).
6. После успешного хода управление перейдет к следующему игроку.

## Структура проекта

- `app/`: Основной пакет приложения Flask.
  - `game/`: Логика игры (движок, данные о плитках).
  - `models.py`: Модели базы данных SQLAlchemy (User, Room).
  - `routes.py`: Маршруты аутентификации и управления комнатами.
  - `sockets.py`: Обработчики Socket.IO для реального времени.
  - `templates/` & `static/`: Фронтенд ресурсы.
- `render_tiles.py`: Скрипт для процедурной генерации плиток с использованием `grafix`.
- `run.py`: Точка входа для запуска сервера.
- `test_*.py`: Набор тестов для проверки логики.
